Welcome to the TMAPS App

TMAPS stands for Time Machine Automated Python Scripts. Time Machine is an interactive viewer developed by the CREATE Lab at Carnegie Mellon University that allows users to explore high resolution datasets along a timelapse. An example of a Time Machine performed by Google on Earth satellite imagery is shown below:




This Tethys app was developed in order to help organize and distribute Time Machines related to hydrologic models. ADHydro is a large-scale, high-resolution watshed simulation developed by the CI-WATER watershed modeling team. It is currently the preview model to display Time Machine capabilities realted to hydrologic modeling. You can find out more information about ADHydro here
