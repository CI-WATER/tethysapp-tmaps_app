Welcome to the TMAPS App

TMAPS stands for Time Machine Automated Python Scripts. The app was developed in order to support the visualization of large transient watershed models. This is the beta version of the app, and the only currently supported model files are the ADHydro output. Stay tuned for additional model support!


This instance of the TMAPS App is setup to be used interactively to for the Admin Users to build Time Machines of ADHydro Output. ADHydro is a large-scale, high-resolution watshed simulation developed by the CI-WATER watershed modeling team. You can find out more information about the project here


The Non-Admin Users of this demo app will be able to view already processed Time Machines of ADHydro output but will not be able to produce their own due to the resources accessed when running the tool.
